# Lydia's Feet

A simple interactive web page to share on WeChat.

## How to Host on GitHub Pages

1. Create a new repository on GitHub
2. Upload the `index.html` file to the repository
3. Go to Settings > Pages
4. Select the main branch as the source
5. Your page will be available at `https://yourusername.github.io/repository-name/`

## Share on WeChat

Once hosted, you can share the GitHub Pages URL directly in WeChat!
